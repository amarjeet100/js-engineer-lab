# Why Conditionals?

Imagine real life.

If it rains → Take an umbrella.

Else → Wear sunglasses.

Programming works exactly the same way.

Conditionals allow your application to make decisions.

Examples:

- Login systems
- ATM Machines
- Netflix Subscription
- Food Delivery Apps
- Banking Applications
- Shopping Websites

Runs code only when a condition is true.

```javascript
if (condition) {
    // code
}
